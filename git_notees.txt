1) Working Dir

2) Staging Area

git add .

3) Local Repo

git commit -m "My First Commit"

4) Remote Repo

git push origin main

5) git log 

6) git status

7) git checkout file_name : used to cancel the changes of working directory

8) git reset HEAD file_name : Used to unstage the file from staging area to working directory. 

9) git branch : Used to list out all branches

10) git branch branch_name : Used to create a new branch

11) git checkout branch_name : used to change to particular branch

12) git checkout -b branch_name : Creates a branch and changes into that branch
